# 🎉 Event Management System

## 📌 Project Overview

The Event Management System is a web-based application developed to manage events and student information efficiently. It provides a simple and user-friendly interface for adding students, creating events, viewing available events, and deleting events. The project demonstrates the integration of frontend, backend, and MongoDB database technologies.

---

## 🚀 Features

- Add student details
- Add new events
- View all events
- Delete events
- User-friendly interface with CSS styling
- MongoDB database integration
- CRUD (Create, Read, Update, Delete) operations

## ⚙️ How to Run the Project

1. Install Node.js and MongoDB.
2. Open the project folder in Visual Studio Code.
3. Open the terminal.
4. Install the required packages:

```bash
npm install
```

5. Start the backend server:

```bash
node server.js
```

6. Open `index.html` in your web browser.

7. Use the application to:
   - Add Students
   - Add Events
   - View Events
   - Delete Events
